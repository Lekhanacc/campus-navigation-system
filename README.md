# Campus Navigation System – IT Block

An indoor navigation system developed for the **IT Block of SJC Institute of Technology**. The project provides an interactive digital map that helps users navigate between rooms and floors within the building.

## Features

- Interactive IT Block floor maps
- Room-level navigation
- Floor-wise navigation
- Shortest path visualization
- Building and floor selection
- User-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
- JSON

## How It Works

1. Open the project in a web browser.
2. Select the IT Block.
3. Choose the desired floor.
4. Select the source and destination rooms.
5. The system displays the shortest navigation path.

## Current Scope

This project currently supports navigation within the **IT Block**.
